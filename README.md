# 🎵 Mood-Based Music Recommendation System

A web-based application that detects user mood using facial emotion recognition and recommends music accordingly via APIs like YouTube and Spotify.

## 🚀 Features

- 🎭 Detects emotions from facial expressions using webcam input
- 🤖 Uses FER (Facial Emotion Recognition) and TextBlob for emotion analysis
- 🎵 Recommends music based on the detected mood
- 🌐 Integrates with YouTube and Spotify APIs for song suggestions
- 🖥️ Built with Flask and OpenCV

## 🧠 Tech Stack

- **Backend:** Python, Flask
- **Emotion Detection:** FER, OpenCV, TextBlob
- **APIs Used:** YouTube API, Spotify API
- **Others:** NumPy, Requests, python-dotenv
